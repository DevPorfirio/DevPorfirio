# 👩🏻‍💻 João Porfirio

**`FullStack Developer`**

I’m currently a developer at CréditoSC Investimentos, a leading company in the buying and selling of government-issued debt securities (precatórios). My goal is to create innovative tech solutions that optimize processes and enhance the efficiency and scalability of the company's systems. I have a solid background in process automation, API development, and systems integration, always focusing on building tools that meet both current and future business needs.

### My professional journey includes:

* Process automation with Python and Selenium, reducing execution time and operational errors.
* API development using Flask to build efficient, scalable backends.
* Systems integration, such as integrating Pipefy, to streamline internal workflows.
* Data and infrastructure management, implementing solutions to optimize system performance and reliability.
* Additionally, I have a strong foundation as an IT Support Technician, with experience in network configuration, user support, and hardware maintenance. 

This background gives me a comprehensive view of systems, from infrastructure to the development layer.
